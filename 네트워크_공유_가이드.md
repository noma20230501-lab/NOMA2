# 부동산 매물 광고 시스템 - 네트워크 공유 가이드

## 🖥️ 서버 컴퓨터 (관리자용)

### 1. 서버 실행 방법

**`서버시작.bat`** 파일을 더블클릭하여 실행하세요.

- 서버 주소: `0.0.0.0` (모든 네트워크 인터페이스에서 접속 허용)
- 포트: `8502`

### 2. 방화벽 설정 확인

Windows 방화벽에서 포트 8502를 허용해야 합니다.

#### 방화벽 설정 방법:

1. **Windows 검색**에서 "고급 보안이 포함된 Windows Defender 방화벽" 검색
2. 왼쪽 메뉴에서 **"인바운드 규칙"** 클릭
3. 오른쪽에서 **"새 규칙..."** 클릭
4. **포트** 선택 → 다음
5. **TCP** 선택, **특정 로컬 포트: 8502** 입력 → 다음
6. **연결 허용** 선택 → 다음
7. 모든 프로필(도메인, 개인, 공용) 체크 → 다음
8. 이름: "Streamlit 부동산 시스템" 입력 → 완료

### 3. 서버 정보

- **서버 IP 주소**: `192.168.219.41`
- **접속 포트**: `8502`
- **접속 URL**: `http://192.168.219.41:8502`

### 4. 서버 유지 관리

- 서버가 실행 중일 때는 명령 프롬프트 창을 **닫지 마세요**
- 서버를 종료하려면 명령 프롬프트 창에서 `Ctrl + C`를 누르거나 창을 닫으세요
- 서버 컴퓨터를 절전 모드나 종료하면 직원들이 접속할 수 없습니다

## 👥 직원용 접속 가이드

### 접속 방법

1. 웹 브라우저(Chrome, Edge, Firefox 등)를 엽니다
2. 주소창에 다음 주소를 입력합니다:

   ```
   http://192.168.219.41:8502
   ```

3. Enter를 누르면 부동산 매물 광고 시스템이 열립니다

### 주의사항

- **같은 네트워크(WiFi/LAN)에 연결되어 있어야 합니다**
- 서버 컴퓨터가 켜져 있고 프로그램이 실행 중이어야 합니다
- 접속이 안 되는 경우:
  - 서버 컴퓨터의 방화벽 설정을 확인하세요
  - 같은 네트워크에 연결되어 있는지 확인하세요
  - 서버 프로그램이 실행 중인지 확인하세요

### 북마크 등록

자주 사용하는 경우 브라우저 북마크(즐겨찾기)에 추가하면 편리합니다:
- `Ctrl + D` (Chrome/Edge)로 북마크 추가
- 이름: "부동산 매물 광고 시스템"

## 🔧 문제 해결

### "사이트에 연결할 수 없음" 오류

1. **서버 컴퓨터 확인**
   - 서버 프로그램이 실행 중인지 확인
   - 명령 프롬프트 창에 오류가 없는지 확인

2. **네트워크 확인**
   - 직원 컴퓨터와 서버 컴퓨터가 같은 WiFi/LAN에 연결되어 있는지 확인
   - 서버 컴퓨터에서 `cmd` 실행 후 `ipconfig` 명령으로 IP 주소 확인

3. **방화벽 확인**
   - Windows 방화벽에서 포트 8502가 허용되어 있는지 확인
   - 회사 네트워크 정책에 의해 차단되지 않았는지 확인

### 느린 속도

- 여러 직원이 동시에 접속하면 속도가 느려질 수 있습니다
- 서버 컴퓨터의 사양(CPU, RAM)을 확인하세요
- 네트워크 속도를 확인하세요

## 💡 추가 팁

### 여러 직원이 동시 작업

- Streamlit은 각 사용자마다 독립적인 세션을 제공합니다
- 한 직원의 작업이 다른 직원에게 영향을 주지 않습니다
- 단, 저장된 파일(피드백 등)은 공유됩니다

### 외부에서 접속 (선택사항)

현재 설정은 **내부 네트워크(LAN)** 전용입니다.
외부 인터넷에서 접속하려면:
- 공유기 포트포워딩 설정
- 공인 IP 주소 확인
- 보안 강화 필요 (HTTPS, 인증 등)
